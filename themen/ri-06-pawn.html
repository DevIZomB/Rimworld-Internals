RI-06 Namespace: Verse

Pawn

Komplexeste Klasse in RimWorld — ~45 Tracker, ~100+ Properties

#1 ~45 Tracker-Felder Verse/Pawn.cs:16-134

Jeder Pawn besitzt ~45 Tracker-Felder (health, jobs, needs, mindState, stances, skills, story, relations, etc.). Jeder Tracker ist ein eigenes Objekt mit eigenem Tick-Zyklus.

#2 Pawn.Tick (Reihenfolge) Verse/Pawn.cs:1525-1603

MP-Kritisch — ~20 Tracker werden in fester Reihenfolge getickt: stances → pather → jobs → health → mindState → needs → etc. Reihenfolge ist gameplay-relevant.

#3 Pawn.TickInterval (Kamera-abhängig) Verse/Pawn.cs:1605-1716

MP-Kritisch — Delta-basiertes Ticking. Interval hängt von der Kamera-Distanz ab (näher = häufiger). Im Multiplayer muss das synchronisiert werden, da verschiedene Spieler verschiedene Kamerapositionen haben.

#5 SpawnSetup Verse/Pawn.cs:1345-1455

Spawn-Logik: Dead-Pawns werden zu Corpse konvertiert, WorldPawns-Liste wird aktualisiert, PawnComponentsUtility initialisiert alle Tracker die noch null sind.

#6 Kill (6 Phasen) Verse/Pawn.cs:2075-2326

MP-Kritisch — 250 Zeilen, 6 Phasen: PreKill-Hooks → Health-State → Corpse-Spawn → Relations-Notify → WorldPawns → PostKill. Rand für Corpse-Rot-Variation.

#7 ~100+ Properties Verse/Pawn.cs:213-800

Über 100 Properties für schnellen Zugriff: IsColonist, IsPrisoner, IsSlave, Faction, Position, Drafted, Downed, Dead, InMentalState, etc.

#8 Temperatur-Verbindung

ComfortableTemperatureRange = Pawn-Stat ± 10°C Puffer (SafeTemperatureRange). Außerhalb davon → Hypothermia/Heatstroke Hediffs. Apparel-Stats modifizieren den Bereich.

Verbindung: → RI-14 (Health/Damage), → RI-22 (Need_Outdoors/Indoors)
